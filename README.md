# interview-task-management

🔷 Interview Task Tracker
A clean and structured React application to manage daily interview preparation with progress tracking and weekly insights.

✨ Features
✅ Add, edit, delete interview tasks
📅 Date-based task tracking
🔁 Rollover unfinished tasks
📊 Weekly summary with stats (completion rate, tech stack, difficulty)
🔒 Past tasks read-only (ensures honest tracking)
⚡ Optimized performance with memoization
🎯 Clean, reusable component architecture

🧱 Tech Stack
- React
- Redux Toolkit
- Tailwind CSS
- React Router

🧠 Key Concepts Implemented
- State management using Redux Toolkit
- Derived data & memoization
- Reusable component design system
- Controlled forms with validation
- LocalStorage persistence
